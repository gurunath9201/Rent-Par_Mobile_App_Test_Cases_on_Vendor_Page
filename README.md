# 🏪 Rent-Par Mobile App – Vendor Page Test Cases

## 📖 Overview

This repository contains manual test cases for the **Vendor Page** of the Rent-Par Mobile Application. The test cases validate the functionality, user interface, and user flow to ensure a smooth vendor experience.

## 📂 Repository Contents

- Vendor Page Test Cases (.xlsx)
- Functional Test Cases
- Positive Test Cases
- Negative Test Cases
- UI Validation Test Cases

- Categories
- Navigation

- Data Validation

- Test Data

- Actual Result
- Status
